Größenangaben

  • px ...... Pixel (1 Ausgabepunkt - abhängig von Ausgabegerät)
  • % ....... Prozent (vom übergeordneten Container abhängig)
  • em ...... abhängig von Schriftgröße
  • rem ..... abhängig von Schriftgröße
  • vh ...... viewport height (orientiert sich an der Höhe des Ausgabebildschirms)
  • vw ...... viewport width (orientiert sich an der Breite des Ausgabebildschirms)

Cascading Order

  • user agent stylesheet (Browser)
  • user normal stylesheet (Usereinstellungen, z.B. Schriftgröße)
  • author normal stylesheet ("normales" Stylesheet)
  • author important stylesheet
  • user important stylesheet
